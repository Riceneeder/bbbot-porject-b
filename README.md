# BBbot Project B

![](./icon.png)

这是一款简单的定时QQ群发工具，是[BBbot Project A](https://github.com/Riceneeder/bbbot-porject-a)的衍伸

## 开发


克隆仓库
```
git clone https://github.com/Riceneeder/bbbot-porject-a.git
```
安装依赖
```
yarn install
```
运行
```
yarn start
```
构建
```
yarn release
```
